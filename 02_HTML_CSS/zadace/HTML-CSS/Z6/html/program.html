Program FPV Akademije

Program je osmišljen kako bi polaznike odveo od početničke razine do sigurnog i samostalnog upravljanja FPV dronovima.

  • Vrste dronova

    Upoznajte različite vrste FPV dronova, njihove namjene te osnovne dijelove svakog FPV sustava.

  • Simulatori

    Prije stvarnog letenja polaznici vježbaju u simulatoru kako bi razvili osjećaj za upravljanje i sigurnost tijekom leta.

  • Elektronika

    Naučit ćete sastavljanje drona, upoznati Flight Controller, ESC, VTX, kameru i baterije te osnove održavanja opreme.
