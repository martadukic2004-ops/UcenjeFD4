Dobrodošli na FPV akademiju

FPV Akademija pruža edukaciju za upravljanje FPV dronovima. Naučit ćete osnove letenja, rad u simulatorima i upoznati elektroniku dronova.

Prijavi se!

O programu

Program FPV Akademije namijenjen je početnicima i svima koji žele naučiti sigurno upravljati FPV dronovima. Nastava uključuje teoriju, rad u simulatoru, upoznavanje elektronike i praktične letove.

Tehničke vještine

Naučite sastaviti, održavati i upravljati FPV dronovima.

Kognitivne sposobnosti

Razvijajte koncentraciju, koordinaciju i brze reakcije.

STEM priprema

Upoznajte elektroniku, tehnologiju i rad FPV sustava.
